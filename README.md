## 1. Information We Collect
- **Account Information:** Email address for authentication.
- **Profile Data:** Age, gender, skin type, lifestyle factors.
- **Facial Photos:** Uploaded for AI skin analysis. Processed
  securely via Xiaomi MiMo API and not permanently stored.
- **Usage Data:** App interaction for improving our service.

## 2. Facial Data Collection and Use

### What Face Data We Collect
Skin Path Guide collects facial photographs that you voluntarily
upload through the app's selfie capture feature.

### How We Use Face Data
Your facial photo is used exclusively for:
- AI-powered skin analysis (hydration, oiliness, acne risk,
  wrinkles, dark spots)
- Generating personalized skincare recommendations
- Creating your skin health profile

### Third-Party Processing
Your facial photo is transmitted to Xiaomi MiMo API
(https://api.mimo.xiaomi.com) for real-time AI analysis.
Xiaomi MiMo processes the photo and returns analysis results.
The photo is processed in real-time and is NOT permanently
stored on Xiaomi's servers.

### Data Retention
- Facial photos are NOT stored permanently on external servers
- Analysis results (numerical scores and recommendations) are
  stored in your account for progress tracking
- You can delete all your data at any time via Settings →
  Delete Account

### Data Security
- All data transmission uses HTTPS encryption
- Facial photos are transmitted securely to Xiaomi MiMo API
- Photos are processed in real-time and immediately discarded
  after analysis

### Your Rights
- You can delete your account and all associated data at any
  time through Settings → Delete Account
- Upon deletion, all your facial data and analysis history
  will be permanently removed
